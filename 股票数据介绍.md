# 股票数据介绍

## 数据概览

本数据集包含中国A股市场的多维度历史数据，涵盖价格、成交量、估值、基本面等多个维度。

## 数据文件列表

| 文件名 | 说明 |
|--------|------|
| `stock_code_info.csv` | 股票代码列表 |
| `day.csv` | 交易日历（日度） |
| `month.csv` | 月度数据日历 |
| `close_day.csv` | 收盘价（日度） |
| `close_adj_day.csv` | 复权收盘价（日度） |
| `amt_day.csv` | 成交额（日度） |
| `turn_day.csv` | 换手率（日度） |
| `pb_lf_day.csv` | 市净率（日度） |
| `float_a_shares_day.csv` | 流通A股数量（日度） |
| `share_totala_day.csv` | 总股本（日度） |
| `cs_indus_code_day.csv` | 中信行业分类代码 |
| `csiall_day.csv` | 中证全指相关数据 |
| `IPO_date_info.csv` | 股票上市日期 |
| `delist_date_info.csv` | 股票退市日期 |

## 数据结构

### 格式说明

大部分数据文件采用**宽表格式**：
- **行**：日期（时间序列）
- **列**：股票代码（例如：000001.SZ, 600000.SH）

### 时间范围

- 起始日期：1998年1月
- 截止日期：2024年11月
- 共约6500+个交易日

### 股票覆盖

- 包含约5600+只股票
- 涵盖A股市场主要股票
- 包含已退市股票信息

## 详细数据说明

### 1. 价格类数据

- **`close_day.csv`**：日收盘价，未复权
- **`close_adj_day.csv`**：日复权收盘价，已考虑分红拆股等因素

### 2. 交易类数据

- **`amt_day.csv`**：日成交额（单位：元）
- **`turn_day.csv`**：日换手率（%）

### 3. 估值类数据

- **`pb_lf_day.csv`**：日市净率（Price-to-Book Ratio）

### 4. 股本类数据

- **`float_a_shares_day.csv`**：流通A股数量
- **`share_totala_day.csv`**：总股本数量

### 5. 基础信息类数据

- **`IPO_date_info.csv`**：每只股票的上市日期
- **`delist_date_info.csv`**：每只股票的退市日期（如未退市则为NaN）
- **`cs_indus_code_day.csv`**：中信行业分类代码

### 6. 指数类数据

- **`csiall_day.csv`**：中证全指相关数据

## 使用建议

1. **复权价格**：进行收益率计算时，建议使用`close_adj_day.csv`
2. **存活偏差**：数据包含已退市股票，可用于避免存活偏差
3. **行业分析**：配合`cs_indus_code_day.csv`可进行行业维度分析

## 注意事项

- 数据中可能存在缺失值（NaN），表示该股票在该日期未上市或已退市
- 股票代码格式：`000001.SZ`（深交所）、`600000.SH`（上交所）
